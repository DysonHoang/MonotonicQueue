Monotonic Queue là gì?

Định nghĩa
Monotonic Queue là một deque (hàng đợi hai đầu) được giữ theo thứ tự đơn điệu — tức là các phần tử bên trong luôn tăng dần hoặc giảm dần từ đầu đến cuối.
🎯 Hình ảnh dễ hiểu
Hãy tưởng tượng bạn đứng xếp hàng mua vé. Nhưng hàng này có quy tắc đặc biệt: khi người mới vào hàng phía sau, tất cả người thấp hơn đứng trước họ bị đuổi ra. Kết quả? Hàng luôn là những người cao dần từ sau ra trước.

Có hai biến thể của Monotonic Queue, dùng cho hai mục đích khác nhau:

Biến thể 1
📈 Tăng dần
Phần tử nhỏ nhất luôn ở đầu deque. Khi thêm phần tử mới, pop ra tất cả phần tử lớn hơn hoặc bằng ở cuối.
→ Dùng tìm MINIMUM
Biến thể 2
📉 Giảm dần
Phần tử lớn nhất luôn ở đầu deque. Khi thêm phần tử mới, pop ra tất cả phần tử nhỏ hơn hoặc bằng ở cuối.
→ Dùng tìm MAXIMUM
💡
Tại sao lưu index thay vì giá trị?
Ta thường lưu chỉ số (index) vào deque thay vì giá trị, bởi vì ta cần biết phần tử đó có còn trong cửa sổ hiện tại không. Nếu index quá cũ → pop ra khỏi đầu deque.

Hoạt động như thế nào?

Với mỗi phần tử mới, ta thực hiện 4 bước theo thứ tự:

1
🗑 Pop front — hết hạn
Nếu phần tử ở đầu deque có index nằm ngoài cửa sổ hiện tại (quá cũ) → bỏ ra khỏi đầu.
2
⚡ Pop back — vi phạm
Pop từ cuối ra tất cả phần tử vi phạm tính đơn điệu so với phần tử mới đang xét.
3
➕ Push back
Thêm index của phần tử hiện tại vào cuối deque.
4
📊 Đọc kết quả
Phần tử ở đầu deque chính là index của min/max trong cửa sổ hiện tại.
🔑
Chìa khóa để hiểu bước 2
Với Monotonic Decreasing (tìm max): Nếu a[cuối_deque] ≤ a[i] → pop cuối deque. Tại sao? Vì phần tử cuối nhỏ hơn, nằm trước phần tử hiện tại, sẽ KHÔNG BAO GIỜ là maximum trong bất kỳ cửa sổ tương lai nào nữa → ta loại bỏ nó.

Cài đặt code

Dưới đây là code đầy đủ cho bài toán Sliding Window Maximum — code ngắn gọn và bám sát 4 bước đã học:

Python
from collections import deque

def sliding_window_max(a, k):
    dq = deque()   # lưu index, giữ thứ tự GIẢM DẦN theo giá trị
    result = []

    for i in range(len(a)):

        # Bước 1: Pop front nếu index nằm ngoài cửa sổ [i-k+1, i]
        while dq and dq[0] < i - k + 1:
            dq.popleft()

        # Bước 2: Pop back nếu phần tử đó nhỏ hơn a[i] (vi phạm giảm dần)
        while dq and a[dq[-1]] < a[i]:
            dq.pop()

        # Bước 3: Thêm index hiện tại vào cuối
        dq.append(i)

        # Bước 4: Đầu deque = index của MAX trong cửa sổ
        if i >= k - 1:
            result.append(a[dq[0]])

    return result

# Kết quả: [4, 4, 5, 9, 9, 9]
print(sliding_window_max([3, 1, 4, 1, 5, 9, 2, 6], 3))
C++
#include <bits/stdc++.h>
using namespace std;

vector<int> slidingWindowMax(vector<int>& a, int k) {
    deque<int> dq;   // lưu index
    vector<int> result;

    for (int i = 0; i < a.size(); i++) {

        // Bước 1: Loại index ngoài cửa sổ
        while (!dq.empty() && dq.front() < i - k + 1)
            dq.pop_front();

        // Bước 2: Loại phần tử nhỏ hơn a[i] (từ đuôi)
        while (!dq.empty() && a[dq.back()] < a[i])
            dq.pop_back();

        // Bước 3: Thêm index hiện tại
        dq.push_back(i);

        // Bước 4: Ghi nhận max
        if (i >= k - 1)
            result.push_back(a[dq.front()]);
    }
    return result;
}
🔄
Muốn tìm Minimum thay vì Maximum?
Chỉ cần đổi dấu so sánh ở bước 2: thay a[dq[-1]] < a[i] thành a[dq[-1]] > a[i]. Deque lúc này tăng dần → đầu deque là MIN.

Monotonic Queue xuất hiện trong rất nhiều dạng bài. Nhận biết qua dấu hiệu chính:

🎯
Khi nào dùng Monotonic Queue?
→ Tìm min/max trong đoạn [i-k, i] khi i duyệt từ trái sang phải.
→ Bài DP có dạng: dp[i] = max(dp[j]) + f(i) với j ∈ [i-k, i-1].
→ Từ khoá: "cửa sổ trượt", "đoạn liên tiếp độ dài k", "không quá k bước".

O(n)

🧮
Tại sao lại là O(n)?
Mỗi phần tử được push vào deque đúng 1 lần và pop ra tối đa 1 lần.
→ Tổng số thao tác = 2n → O(n).
Không có vòng lặp lồng nhau thực sự — while bên trong chạy tổng cộng O(n), không phải O(n) mỗi lần.
📝
Tóm tắt toàn bộ
Monotonic Queue = deque + tính đơn điệu.
Công thức vàng: pop front (hết hạn) → pop back (vi phạm) → push back → đọc front.
Dùng khi cần min/max trong cửa sổ trượt hoặc tối ưu DP có dạng lấy max/min trên đoạn.
